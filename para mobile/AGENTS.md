# Instruções da Documentação Mobile

As instruções canônicas do projeto estão no `AGENTS.md` da raiz.

Ao editar esta pasta:

- preserve a hierarquia definida no arquivo raiz;
- mantenha `00-contexto-operacional.md` curto e atual;
- registre mudanças arquiteturais em `06-registro-decisoes.md`;
- diferencie claramente contratos existentes, planejados e dependentes;
- não duplique especificações completas entre arquivos;
- atualize o `README.md` quando adicionar, remover ou renomear documentos.
